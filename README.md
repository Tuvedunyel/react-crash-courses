# React crash courses

Dépôt du cours sur React pour débutant, version 2021.

## Dépendencies

- React (version 17.0.2)
- React Dom (version 17.0.2)
- Sass (version 1.49.7)
- vite (version 2.8.0)

## Installation

```bash
git clone https://github.com/Tuvedunyel/react-crash-courses.git
cd react-crash-courses
npm install
```

## Command Line

Live Sever

```bash
npm run dev
```

Build

```bash
npm run build
```
